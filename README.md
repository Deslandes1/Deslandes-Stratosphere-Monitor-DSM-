# 🔴 Deslandes Stratosphere Monitor (DSM)
### *Global Airspace & Orbital Surveillance Suite*

**Deslandes Stratosphere Monitor (DSM)** is a professional-grade tactical tracking application designed for monitoring civilian aviation, orbital satellite positions, and high-velocity ballistic threats. Developed by **Gesner Deslandes**, this suite provides a unified "OpenSky" interface for technology researchers and defense simulation.

---

## 📡 Core Monitoring Modes

### 🚀 1. Missile Detector (Tactical Mode)
* **Dual-Needle Radar:** A specialized high-contrast polar interface with 180-degree opposing sweep needles for real-time threat detection.
* **Threat Classification:** Categorizes ballistic objects into **ICBM**, **Cruise**, and **Hypersonic Glide Vehicles**.
* **Impact Prediction:** Automated math engine calculates estimated impact coordinates based on current heading and Mach velocity.
* **Intelligence Export:** Generate and download localized `.txt` reports of all detected stratospheric threats.

### ✈️ 2. Aircraft Radar
* **ADS-B Surveillance:** Integrated logic for tracking civilian and unscheduled flights using OpenSky Network protocols.
* **High-Altitude Monitoring:** Specifically tuned to monitor traffic within the upper troposphere and lower stratosphere.

### 🛰️ 3. Satellite Tracker
* **Orbital Pathing:** Visualizes satellite ground tracks and predicts future passes over specific geographic coordinates.
* **TLE Integration:** Designed to process Two-Line Element sets for precise orbital positioning.

## 🛠️ Technical Architecture

* **Language:** Python 3.10+
* **Framework:** Streamlit (Web UI)
* **Graphics:** Plotly Graph Objects (Polar & Geographic)
* **Deployment:** Optimized for GitHub & Streamlit Cloud

### Installation
1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/YOUR_USERNAME/Deslandes-Stratosphere-Monitor.git](https://github.com/YOUR_USERNAME/Deslandes-Stratosphere-Monitor.git)
    ```
2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **Launch Interface:**
    ```bash
    streamlit run app.py
    ```
